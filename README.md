# Jarvis

A Jarvis assistant with Ollama fueling a modification on the Gemma 3 model by Google. The assistant consist of a pipeline merging a couple different libraries and models.

Speech to text - faster_whisper
LLM - Ollama with a modified gemma 3 model
Text to speech - Piper TTS with a jarvis voice model

Wakeword - Openwakeword
Voice Activity Detection - Silero VAD
